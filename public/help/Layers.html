Setting visualisation properties ››

Map layout composition

This panel provides access to all layers which can be displayed in the map window. Each layer could be switched on or off by checking / unchecking the check box in front of the name of the layer.


This panel provides two cards with different modes of layers listing:

  • card with all available layers
  • card with layers selected for visualization
