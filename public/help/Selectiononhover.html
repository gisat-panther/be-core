Selection of analytical unit on hover

This interactive way of unit selection is enabled by activation of hover tool in map tools panel. If the hover tool is activated, the user can interactively select the unit by hover the mouse over it in map, tree of analytical units or in any chart. Please be aware of the fact that this is only temporal selection, which changes with every move of mouse over other analytical unit.
