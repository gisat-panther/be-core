Zooming to unit of interest

Map tools ››

Zoom to selected units

This tool enables zooming to analytical units selected in the frame of active selection. Active selection is defined by the color selected in the multiple selection panel.


User specified rectangle

Another possibility how to zoom into the unit of user's interest is to draw a user specified rectangle directly in the map window:

This can be done by pressing the SHIFT button and holding it down, while drawing a rectangle on the map, which specifies the area of user's interest.
