Map tools

The map tools panel provides the most important of the standard map controls, which are necessary for map navigation, zooming in/out and measuring distances or areas.


The following functionalities are available in the map tools panel:

  • Hover
  • Zoom to selected units
  • Measure line
  • Measure polygon
  • Multiple maps mode activation
  • Save map area as image
