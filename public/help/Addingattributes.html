Adding attributes

To add attributes to the list, press the “add” button over the table header. Then select attributes you want to add by checking their checking box. You can use ascending or descending sorting, as well as filtering by text string for both attributes and attribute sets.

For visualizing selected attributes in corresponding chart directly from the “add attribute” dialog, press “configure” button under the table. However, in such case, the chart attribute list will not be modified and the attribute selection is only temporal.

If you decide to return to the initial chart attribute list without modification, use the “Back” button.

For adding selected attributes to the chart attribute list, it is necessary to press the “add” button under the table. The dialog returns to the chart attribute list mode. Clicking the button “configure” under the table in this mode, corresponding chart redraws and all attributes from the list will be visualized it the chart.
