Normalization by attribute set

To normalize values of attributes by attribute set, select "Attribute set" in the Norm type menu.

Then, specify the attribute set for normalization and click the "Change normalization" button at the bottom of the card.

After that, normalization by "Attribute set" will be indicated in the "Normalization" column in the chart configuration dialog:


Normalization by attribute set is a proper solution for special cases, when you need to display the share of more different attributes as regards more different (and corresponding) attributes. A typical example is displaying percentages of land cover formation by class (the amount of land cover formation of each class is normalized on the initial area of corresponding land cover class):
