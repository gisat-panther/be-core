Setting analysis characteristics – extended mode of the panel

This version of the analysis characteristics settings panel lets you specify details of the analysis.


As well as in the case of basic mode of the panel, you can specify basic analysis characteristics by setting the following items:

  • Scope of analysis
  • Theme of interest
  • Place of interest 


Compared to the basic mode of this panel, the extended mode provides also the following possibilities to modify the analysis characteristics:

  • switching the year of interest
  • switching the analysis level
  • selecting the visualization in the visualization menu


This mode of the panel is always available at the top of the analysis window, so you can modify the analysis characteristics anytime while working with the application.
