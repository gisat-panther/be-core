Setting analysis characteristics  - extended mode of the panel

This version of the analysis characteristics settings panel provides the user a possibility to specify the details of analysis.


As well as in case of basic mode of the panel, the user can specify basic analysis characteristics by setting following items:

  • The scope of analysis
  • Theme of interest
  • Territory of interest 


Compared to the basic mode of this panel, extended mode provides also following possibilities to modify the analysis characteristics:

  • switching the year of interest
  • switching analysis level
  • selecting the visualization in visualization menu


This mode of the panel is always available at the top of the analysis window, so that the user can modify the analysis characteristics anytime while working with the application.
