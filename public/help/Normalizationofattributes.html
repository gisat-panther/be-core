Normalization of attributes

To enable the user to compare the values at different scales, a possibility of attribute normalization is provided. Attributes to be normalized have to be selected by checking their check boxes in the chart configuration dialog. After that, open the “Normalize” card and select the normalization type.


PUMA platform provides following normalization types:

  • normalization by area
  • normalization by attribute
  • normalization by attribute set
