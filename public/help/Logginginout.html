Logging in / out

To get a possibility of saving and sharing data view modifications, you have to log in using an assigned user name and password.

Users with accounts already created can log in to the application by clicking the "login" button in the upper right corner and then filling in the form with a user name and password:

Users with no active account must first sign up, using the "sign up" button, filling in their chosen user name, e-mail address and password.

After logging in, a data view panel appears. After finishing the work, you should log out from the application using the "logout" button.
