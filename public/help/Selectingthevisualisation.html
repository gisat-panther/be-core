Selecting the visualisation

The menu provides possibility to select from the list of predefined visualizations. The user is also provided by possibility to create and save his own visualizations.


Visualisations could be modified via:

  • setting the charts´ properties in the chart panel
  • setting other visualization properties in the left panel
