Sorting analytical units

The application provides a possibility of sorting analytical units based on their indicator values.

This sorting is enabled in the tables in the chart panel. It is possible to sort the table by sorting values in any column in ascending or descending mode. Sorting is activated by single click on the column header.

Actual sorting of the table is immediately projected also in column charts, where corresponding attributes are sorted in the same way.
