Exporting charts as table

Data used in any chart in the chart panel can be exported from the application in CSV format by clicking the “Export as CSV” button . All tables can also be exported this way. Values for all units at the most detailed level displayed in the chart/table will be exported into the CSV format (not only the limited number of units that can be displayed in a chart in one page).
