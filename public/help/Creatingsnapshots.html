Snapshot tool

The Snapshot tool enables to create snapshot of actual map layout. Modify the map layout according to your requirements using tools incorporated in the left section {#leftSection} and click the “Snapshot” button. The snapshot of the map view is created and it appears in the snapshot section of the chart panel.
