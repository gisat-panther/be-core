Chart attribute list

For each chart, several attributes from full attribute list {#attListFull} are selected. To view or modify the list of chart attributes, click the button for chart configuration above the chart which should be modified.


The attribute list can be sorted ascending or descending. You can switch between “descending” and “ascending” mode by clicking on the head row of the table column. Each attribute belongs to an attribute set  – corresponding sets are listed in the second column of the table. Attribute sets could also be sorted ascending or descending in the list. Both attribute and attribute sets could be filtered by key text string using the attribute filter hidden on the right side of the header cells of the table.
