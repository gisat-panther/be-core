Regional scope: AsiaPop population data for countries in EAPAC region


This scope provides lowest level of detail and largest area. In frame of this scope, only population data come into analysis.

Aggregated data incorporated in application have been calculated using zonal analysis based on source population rasters from AsiaPop dataset.

Not all countries of East Asia and Pacific region are covered by aggregated data yet, the rest of EAPAC region will be added to the dataset.
