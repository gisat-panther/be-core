Selection of analytical units

The user can select arbitrary analytical units of his interest and compared them between each other. Analytical units at different hierarchical level can be also compared.


There are tree ways how to select the units:

  • Selection in hierarchical tree of analytical units
  • Selection in map
  • Selection in charts


For multiple selection hold CTRL key during the selection – multiple selection is enabled in all hierarchical tree, map and charts.

There is also a possibility to make more multiple selections utilizing the selection by multiple selection tool and compare between these groups of analytical units.

Selected units will be highlighted by corresponding color in all hierarchical tree of units, map and all charts in the same time.


There is also a possibility of highly interactive temporal selection on mouse hover.


Advanced possibilities for units´ selection provides the advanced filter tool.
