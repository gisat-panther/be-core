Setting analysis characteristics  - basic mode of the panel

Basic version of this panel is intended for accessing the analysis tool. It is necessary to fill in all tree menus in the panel to open the analysis.  


In the basic mode of the panel, the user can specify basic analysis characteristics by setting:

  • scope of analysis
  • theme of interest  
  • territory of interest


You have to select the item in all three menus and  press the “Confirm” button to get to the next session. This confirmation expands the panel for setting of analysis characteristics  to the extended mode it also opens the default visualization for selected combination of scope and theme and zooms to territory of interest.
