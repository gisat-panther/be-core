Filtering analytical units by indicator values (Advanced filters)

This tool provides an advanced possibility to filter analytical units by their attributes values. It enables to limit the number of displayed analytical units via setting user defined interval of indicators´ values. Only analytical units with indicator values in defined interval are displayed in charts and map.


For setting desired intervals for selected indicators, expand the advance filters panel and click the button “Configure”. Select attributes by which you want to filter the units on “Add” card via checking their check boxes and add them to the attribute list for filtering using the “Add” button. You can also set a normalization conditions for each attribute using the card “Normalize”. When the attribute list for normalization is complete press the “Configure” button. It adds filters for each attribute from the list to the advance filters panel.

These filters enables the user to define intervals for values of corresponding indicators for which analytical units should be included into analysis.
