Reference Satellite Images

This group of layers contains satellite imageries that were used as reference source data for Land Cover and Change mapping of metropolitan regions in the frame of PUMA mapping production carried out by GISAT. Therefore they are relevant and available only for local scope in the frame of this project.


Two types of satellite imageries are presented here:

- Reference satellite imagery – used as reference data source for mapping

- Alternative reference imagery – used as alternative data source for mapping in areas where reference imagery was impacted by cloud coverage


Data from 4 different sensors, all of them of high spatial resolution and multispectral band composition, have been used as reference (or alternative) satellite imageries. Details on these sensors are provided in the table below:
