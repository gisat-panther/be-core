PUMA project

The PUMA project has been contracted by the World Bank to Czech Republic based GISAT company. The main aim of this project has been to develop a Platform for Urban Management and Analysis (PUMA).  

Beside this software development, the PUMA project also involves satellite Imagery based mapping. Land cover and change in five metropolitan areas in eastern and south eastern Asia has been mapped in frame of this project. Details on this mapping are specified in the Interim report of the PUMA project.

More information on the PUMA project is available on the project website.
